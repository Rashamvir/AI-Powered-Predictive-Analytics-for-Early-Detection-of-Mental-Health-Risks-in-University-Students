# 🧠 AI-Powered Predictive Analytics for Early Detection of Mental Health Risks

### 📍 Project Goal
This project builds a machine learning system that predicts **mental health risk levels (Low / Medium / High)** among university students using behavioral, academic, and lifestyle data.  
The primary objective is to enable **early intervention** by universities, counselors, and wellness support teams.

---

## ✅ Key Features

| Module | Description |
|--------|-------------|
| 🔍 Data Preprocessing | Handling missing values, encoding categorical data, feature scaling |
| 📊 Exploratory Data Analysis (EDA) | Correlation heatmaps, risk distribution graphs |
| 🤖 Multi-Model Training | Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, XGBoost |
| ⚖️ SMOTE for Class Balance | Synthetic Minority Oversampling Technique to handle class imbalance |
| 🏆 Model Comparison | Accuracy, F1-score, confusion matrix, visual comparison charts |
| 🧠 Explainability | Feature importance and SHAP analysis (optional) |
